Image Data Regeneration
Using Machine Learning

About

The goal of the project was to explore how well Machine Learning techniques can regenerate data lost through JPEG image compression. It was completed as part of our final year design project requirements.

We were able to evaluate two Machine Learning models, a Generative Adversarial Network (GAN) and a Convolutional Neural Network (CNN). The GAN model exhibited better performance, and was selected as the final model.
